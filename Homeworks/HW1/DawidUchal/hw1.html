Task 1¶

Assume:

$A$ - being a part of a given group: $a$ - Blue (privileged), $b$ - Red (unprivileged)

$Y$ - whether will ($Y = 1$) or won't ($Y = 0$) use XAI

$\hat{Y}$ - decision about being enrolled ($\hat{Y} = 1$) or not ($\hat{Y} = 0$)

As half of the Red group will be allocated randomly to training, and half of the group will find the skills useful, we can assume that all possible scenarios for that group will have equal probability of 25%.

  1. Demographic parity:

    • Blue: $P(\hat{Y} = 1|A = a) = \frac{65}{100} = 0.65$

    • Red: $P(\hat{Y} = 1|A = b) = \frac{50}{100} = 0.5$

    • ratio: $\frac{P(\hat{Y} = 1|A = a)}{P(\hat{Y} = 1|A = b)} = \frac{65}{50} = 1.3 $

  2. Equal opportunity:

    • Blue: $P(\hat{Y} = 1|A = a, Y = 1) = \frac{60}{80} = 0.75$

    • Red: $P(\hat{Y} = 1|A = b, Y = 1) = \frac{25}{50} = 0.5$

    • ratio: $\frac{P(\hat{Y} = 1|A = a, Y = 1)}{P(\hat{Y} = 1|A = b, Y = 1)} = \frac{0.75}{0.5} = 1.5 $

  3. Predictive rate parity:

    • Blue:

      • $P(Y = 1|A = a, \hat{Y} = 1) = \frac{60}{65} \approx 0.92$

      • $P(Y = 0|A = a, \hat{Y} = 0) = \frac{15}{35} \approx 0.43$

    • Red:

      • $P(Y = 1|A = b, \hat{Y} = 1) = \frac{25}{50} = 0.5$

      • $P(Y = 0|A = b, \hat{Y} = 0) = \frac{25}{50} = 0.5$

    • ratios:

      • $\frac{P(Y = 1|A = a, \hat{Y} = 1)}{P(Y = 1|A = b, \hat{Y} = 1)} = \frac{0.92}{0.5} = 1.84 $

      • $\frac{P(Y = 0|A = a, \hat{Y} = 0)}{P(Y = 0|A = b, \hat{Y} = 0)} = \frac{0.43}{0.5} = 0.86 $

In all of the above cases the ratios are higher than 125% (except for the negative predictive parity), meaning that the Blue group is priviliged.

Starred task: How can this decision rule be changed to improve its fairness?

To improve the fairness of the above decision rule, we can use a better model (not random, but based on some other prior information) for assinging people from Red group, thus increasing the number of accuratelly allocated people.

def preprocess(df):
    df.columns = ['age', 'workclass', 'fnlwgt', 'education', 'education_num', 'marital_status', 'occupation', 'relationship', 
                 'race', 'sex', 'capital_gain', 'capital_loss', 'hours_per_week', 'native_country', 'income']
    df = df.replace(' ?', np.nan)
    df = df.dropna()
    
    y = df['income'].apply(lambda x: 0 if x == ' <=50K' else 1)
    df = df.drop('income', axis=1)

    cat_columns = ['workclass', 'education', 'marital_status', 'occupation', 'relationship', 'race', 'native_country', 'sex']
    X = pd.get_dummies(df, columns=cat_columns)

    return X, y
